Para ejecutar este programa hay que tener unas cuantas consideraciones:
1. instalar graphlab (como se vio en clase)
2. al intalarlo este instalará anaconda 2, así que abriremos la consola de anaconda para graphlab anaconda prompt (gl-env)
3. todo lo realizaremos desde esta consola
4. instalaremos los siguientes paquetes usando el comando pip install "NOMBRE-PAQUETE":
	scikit-fuzzy
	openpyxl
5. luego de esto nos ubicaremos en la carpeta del proyecto usando el comando cd #Ubicacion-LOCAL#/IA-G1
6. ejecutaremos el codigo usando el comando python main.py
